# Portfolio

My personal developer portfolio built with Next.js. Shows my projects, skills, experience, and how to reach me.

## Tech Stack

- Next.js (App Router)
- CSS (custom, no framework)
- JavaScript

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Projects Featured

- **PantherPool** - full-stack rideshare platform with Firebase Auth, PostgreSQL, and Prisma
- **SmartCart** - grocery store navigation and coupon app built as a SaaS concept
- **Harmonize** - music-based dating app using the Spotify API
- **Darazi Travels** - travel agency website with an AI chatbot powered by Groq and Llama 3.1
- Plus several Python and automation projects
